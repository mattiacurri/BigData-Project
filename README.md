## Project Organization

```
├── Makefile
├── pyproject.toml             <- Project configuration with package metadata
├── README.md                  <- The top-level README for developers
│
├── docs                       <- Documentation (mkdocs project)
│
├── notebooks                  <- Jupyter notebooks for analysis
│   ├── 1-DescribeData.ipynb   <- Data description and exploration
│   └── 2-DataQuality.ipynb    <- Data quality analysis
│
├── references                 <- Data dictionaries and explanatory materials
│
├── reports                    <- Generated analysis (LaTeX, figures, etc.)
│   ├── businessunderstanding.tex
│   ├── conclusion.tex
│   ├── datapreparation.tex
│   ├── dataunderstanding.tex
│   ├── deployment.tex
│   ├── evaluation.tex
│   ├── modeling.tex
│   ├── main.tex
│   ├── main.bib
│   ├── figures/
│   ├── notes.md
│   └── presentation/
│       └── beamerthemeSimplePlus.sty (+ style files)
│
└── src                        <- Source code for the project
    ├── __init__.py            <- Makes src a Python module
    └── modeling/              <- Modeling utilities
```
